# Salt Identification on Seismic Data using CNN

## Overview
This project demonstrates a deep learning approach for identifying salt bodies in seismic images using a Convolutional Neural Network (CNN).

## Dataset
- TGS Salt Identification Dataset (Kaggle)
- Grayscale seismic images (101Ã—101)
- Binary salt masks

## Tools
- Python
- PyTorch
- OpenCV
- NumPy
- Matplotlib
- Google Colab (GPU)

## Method
- Image preprocessing (resize, normalization)
- CNN-based image segmentation
- Binary Cross Entropy loss
- Adam optimizer
